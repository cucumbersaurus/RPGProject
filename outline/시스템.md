# 시스템

## 기초 게임 시스템

- 강화 시스템
- 스탯 시스템
    - 스탯 종류
        
        근력 : 공격력 증가
        
        민첩 : 공격속도 증가
        
        신속 : 이동속도 증가
        
        체력 : 초당 체력 회복량 증가
        
        내구 : 기본 방어력 증가
        
        행운 :  운이 좋아지는 기분이 든다
        
        손재주 : 치명타 확률 증가
        
        매력 : 아이템 구매가 감소 , 몬스터의 인식 범위감소
        
    - 스탯 포인트
        
        레벨을 올릴때마다 1씩 지급
        
        5레벌마다 1씩 추가 지급 
        
    - 스탯 초기화
